# 学词宝 📚

英语单词 + 语文生字学习工具，适配小学四年级至初中一年级。

## 功能

- **英语单词**：PEP 教材同步，四至七年级共 ~330 词
- **语文生字**：人教版同步，四至七年级共 ~280 字
- **学习模式**：看词回想，点击揭晓释义
- **测验模式**：四选一，检验掌握程度
- **错题复习**：自动记录不会的词，集中练习
- **发音功能**：英语单词支持朗读
- **进度跟踪**：学习数据保存在浏览器中
